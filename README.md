# BMW-Used-Car-Dashboard
Power BI dashboard analyzing BMW used car market 1996-2020

 📊 Overview
An interactive Power BI dashboard analyzing 10,664 BMW used car listings 
across models, fuel types, transmission types and price trends from 1996 to 2020.

 🔍 Key Insights
- Diesel dominates the market at 65.5% of all listings
- Average BMW used car average price grew 490.5% from 1996 to 2020
- 3 Series is the most listed model with 2,434 cars (22.8% market share)
- Electric vehicles have the best fuel efficiency at 470.8 MPG

 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)

 📌 Dashboard Features
- KPI Cards: Average Price, Total Cars, Average Mileage, Top Model
- Line Chart: Average price trend by year
- Column Chart: Average price by model
- Bar Chart: Total cars by fuel type
- Donut Chart: Transmission breakdown
- Bar Chart: Average MPG by fuel type
